# My dotfiles

This is my working in progress personal dot files

## dependencies

### Mac

```bash
brew install gd fzf zoxide make fzcmake
```

### Linux

```bash

sudo apt install git nvim fzf tmuxi vfd-find

# install rust first

cargo install --locked fzf-make
```
